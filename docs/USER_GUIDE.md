# ユーザーガイド

## はじめに

NeuralForecast AutoML WebUIへようこそ！
このガイドでは、システムの基本的な使い方から高度な機能まで、
ステップバイステップで説明します。

## 基本的な使い方

### 1. データの準備

時系列データは以下の形式が必要です：

```csv
unique_id,ds,y
series_1,2024-01-01,123.45
series_1,2024-01-02,125.67
series_2,2024-01-01,234.56
...
```

必須カラム：
- `unique_id`: 系列の識別子
- `ds`: 日時（datetime形式）
- `y`: 目的変数（数値）

### 2. データのアップロード

1. 左サイドバーから「📤 Data Upload」を選択
2. ファイルをドラッグ&ドロップまたはブラウズ
3. 検証結果を確認
4. 「Proceed」をクリック

### 3. モデルの選択と設定

#### Quick Startモード（初心者向け）
- プリセットから選択：Balanced / Fast / Accurate

#### Standardモード（中級者向け）
- モデル選択: NHITS, TFT, DLinearなど
- 基本パラメータ設定
- リソース配分

#### Advancedモード（上級者向け）
- すべてのハイパーパラメータを手動設定
- カスタム探索空間の定義

### 4. 学習の実行

1. 「🚀 Training」ページへ移動
2. 設定を確認
3. 「Start Training」をクリック
4. リアルタイムで進捗を監視

### 5. 結果の確認

「📈 Results」ページで以下を確認：
- 予測グラフ
- 性能指標（RMSE、MAE、MAPEなど）
- 最適パラメータ
- リソース使用状況
